# Latest Research Trends in AI

Build the English and Korean papers with XeLaTeX so Korean text, TikZ figures, and PGFPlots charts render correctly:

```bash
latexmk main.tex
latexmk main_ko.tex
```

The repository includes `.latexmkrc`, so `latexmk` automatically uses `xelatex` and `bibtex`.
